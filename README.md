## How to Use

1. Open any code file (e.g. `.ts`, `.js`, `.py`).
2. Look for the **Structura** icon in the Activity Bar (List icon).
3. Drag any function or class to reorder it within the file.
4. Done! Code is automatically moved and spacing is preserved.

## Requirements

- VS Code 1.74.0 or higher

## Known Limitations

- MVP version: Best for flat function reordering.
- Formatting: Basic whitespace handling is included, but running "Format Document" after large moves is recommended.
